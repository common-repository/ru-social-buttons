Sharing articles to social networks
== Description ==
Данный плагин позволяет вашим читателям публиковать ссылки и делиться материалами в
социальных сетях(Twitter, Facebook, Вконтакте(vkontakte.ru), Мой мир)

Sharing articles to twitter, facebook, vkontakte and mail.ru using original buttons.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
